# スレ詠みちゃんChrome拡張

## 概要

スレ詠みちゃんのChrome拡張はimg/aimgのスレッドから直接スレ詠みちゃんを起動する機能を持ちます。  
旧tegaki_saveとはレイアウトが競合するため併用はできません。


## インストール手順

スレ詠みちゃんの解凍フォルダに存在する**chrome-extensionsフォルダ**を開き登録します。

1. Chromeを起動しアドレスバーに **chrome:\/\/extensions\/** を入力して管理画面を開きます

2. 有効にしていない場合デベロッパーモードを有効にします　
![-](/images/usage/chrome-extensions_010_devmode.png)

3. パッケージ化されていない拡張機能を読み込むボタンが表示されるので押下してフォルダを登録します
![-](/images/usage/chrome-extensions_011_install.png)


また[スレ詠みちゃんの設定画面から連携](./settings#chrome-extensions)も設定してください。


## 機能

### 開始  
現在のスレッドを最初から読み上げします

### 最終レスから開始
現在のスレッドの最後のレスから読み上げを開始します


![-](/images/usage/chrome-extensions_020_image.png)
