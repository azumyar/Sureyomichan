# よくある質問

## ツール起動時にメッセージが出る

以下のメッセージが表示される場合があります。  
これはソフトを動かす為の.NET 10というMicroSoftが提供しているアプリを動かす為のフレームワーク（材料みたいなもの）です

```

[Window Title]
Sureyomichan.exe

[Main Instruction]
You must install or update .NET to run this application.

[Expanded Information]
Required: 'Microsoft.NETCore.App', version '10.0.0' (x64)

Learn more:
https://aka.ms/dotnet/app-launch-failed

Download link:
https://aka.ms/dotnet-core-applaunch?framework=Microsoft.NETCore.App&&framework_version=10.0.0&&arch=x64&&rid=win10-x64&&gui=true

[V] 詳細の表示(D)  [Download it now] [閉じる(C)]`


```

メッセージにある以下のリンクから.NET 10をインストールお願いします。  
※リンクを開くと自動的に.NETのインストーラーがダウンロードされます。  

https://aka.ms/dotnet-core-applaunch?framework=Microsoft.NETCore.App&&framework_version=10.0.0
