# インストール・アンインストール手順

## インストール手順


ダウンロードページ（ https://harukei66494739.github.io/Sureyomichan/ ）  
よりダウンロード(v.yyyymmdd)からファイルをダウンロードしてください。  
ダウンロードされるファイルは**sureyomichan-v.yyyymmdd.zip**という圧縮ファイルになります。  
適当な場所で解凍（解凍ソフトは7zip推奨）してください。
