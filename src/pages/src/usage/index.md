# スレ詠みちゃんについて

スレ詠みちゃんについては某所対応の配信用スレッド読み上げソフトウェアです。tegaki_saveの後継として開発されています。


## 動作環境

|種別|バージョン|
|----|----|
|OS|Windows 11|
|そのほか|.NET Desktop Runtime v10|

::: warning 注意！
ARM64アーキテクチャは環境がないので確認していません。  
64bit版Windows 10はおそらく動作すると思いますが確認していません。32bit版では動作しません。
:::
